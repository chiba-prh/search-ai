## 1. OpenAI、GPT-5.3 Instant Miniをリリース — 新しいフォールバックモデル
*GPT-5.3 Instant Mini released in ChatGPT*

| Field | Value |
|---|---|
| **Source** | OpenAI Blog |
| **URL** | [https://openai.com/news/](https://openai.com/news/) |
| **Published** | 2026-04-10 |
| **Importance** | ⭐⭐⭐ (3/5) |

2026年4月、OpenAIはChatGPTにおける新しいフォールバックモデル「GPT-5.3 Instant Mini」をリリースした。このモデルはレート制限に達したユーザー向けに、従来のGPT-5 Instant Miniに代わって自動的に切り替わるモデルとして機能する。GPT-5.3 Instant Miniはより自然な会話能力、強化されたライティング性能、改善されたコンテキスト理解を備えている。また、OpenAIは新たに月額100ドルのProプランを導入し、Codexの利用方法をPlusおよびProティアで見直した。

- GPT-5 Instant Miniに代わる新フォールバックモデルとしてChatGPTに搭載
- 自然な会話、ライティング性能、コンテキスト理解が前世代より向上
- 新たに月額100ドルのProプランを導入、Codex利用体系も刷新

## 2. BloClaw — 次世代科学的発見のためのマルチモーダルAIワークスペース
*BloClaw: An Omniscient, Multi-Modal Agentic Workspace for Next-Generation Scientific Discovery*

| Field | Value |
|---|---|
| **Source** | arXiv (cs.AI / cs.LG) |
| **URL** | [https://arxiv.org/list/cs.AI/current](https://arxiv.org/list/cs.AI/current) |
| **Published** | 2026-04-10 |
| **Importance** | ⭐⭐⭐ (3/5) |

2026年4月、arXivにて「BloClaw」と題されたマルチモーダルAIエージェントワークスペースに関する論文が発表された。BloClawは科学的発見を加速するために設計された全知型のエージェントシステムで、テキスト・画像・データなど複数のモダリティを統合的に処理する。従来の単一タスク型AIツールとは異なり、仮説生成から実験設計、データ分析までの科学研究プロセス全体をエンドツーエンドで支援することを目指している。

- マルチモーダル対応のAIエージェントワークスペースを提案
- 仮説生成から実験設計・データ分析まで科学研究の全プロセスを支援
- 次世代の科学的発見を加速するエンドツーエンドのアプローチ

`Tags:` arxiv, multi-modal, scientific-discovery, agents, research

## 3. PsychAgent — 経験駆動型の生涯学習AIカウンセラー
*PsychAgent: An Experience-Driven Lifelong Learning Agent for Self-Evolving Psychological Counselor*

| Field | Value |
|---|---|
| **Source** | arXiv (cs.AI / cs.LG) |
| **URL** | [https://arxiv.org/abs/2604.00931](https://arxiv.org/abs/2604.00931) |
| **Published** | 2026-04-10 |
| **Importance** | ⭐⭐⭐ (3/5) |

2026年4月、arXivにてAI心理カウンセリングを生涯学習問題として定式化した初のフレームワーク「PsychAgent」が発表された。記憶拡張型計画エンジン（Memory-Augmented Planning Engine）で長期的な一貫性を維持し、スキル進化エンジンが過去のカウンセリング軌跡から治療スキルを抽出する。GPT-5.4やGemini-3といった汎用LLMを上回るスコアを達成し、特にDialogue PlanningやCTRS指標で大幅な改善を示した。静的なファインチューニングを超え、蓄積された経験から継続的に成長するAIカウンセラーを実現している。

- AI心理カウンセリングを経験駆動型の生涯学習問題として初めて定式化
- GPT-5.4やGemini-3を上回るカウンセリング性能を達成
- 記憶拡張型計画エンジンとスキル進化エンジンの組み合わせで継続成長

`Tags:` arxiv, mental-health, lifelong-learning, agents, counseling
